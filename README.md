# Online Food Ordering System (Spring Core)

Its a console based Application which is made just to implement Spring concepts.

## Concepts Covered
@Configuration, @Component, @Bean, @Primary, @Qualifier, @PropertySource, @Value, @Scope, @PostConstruct, @PreDestroy, ApplicationContext

## Tech Stack
Java, Maven, Spring Core, VScode

## How to Run
Install Maven and setup the environment variables, then clone the repository and hit

>mvn compile

Then run OfosMain.java from your IDE.
